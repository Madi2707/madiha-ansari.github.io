Project Overview

The project focuses on developing a wayfinding system that tracks live audits of 26,000 bus stops accross Victoria. The system utilizes advanced data engineering techniques to handle GPS data. This was built for Department of Transportation.

Daily audits tracking provided insights on:

  • Maintenance.
  • Accurate Bus Stop location capture.
  • Identifying issues with Bus Stop Infrastructure.

The main idea was to engineer data integration for different data sources inorder to produce a single visual that had the ability to present user friendly live data.

Conclusion

The Wayfinding system represents a significant advancement in navigation technology, combining real-time data processing with intelligent algorithms to enhance user experience and efficiency in urban environments. The project demonstrates the potential of innovative solutions in improving everyday navigation and accessibility.
