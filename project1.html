Project Overview

The project focuses on developing a system that intelligently monitors and provides data driven analytics for bus accidents in urban environment. By leveraging advanced machine learning models and GPS data processing, the system aims to enhance road safety and provide actionable insights for fleet operations.

This Dashboard visualizes locations with the frequency and severity of accidents on different routes. I used DBSCAN Clustering Algorithm to identify route clusters.

The daily monitoring Dashboard provides details of daily accidents. It displays accident locations, routes, and severity, enabling quick responses from emergency services or operational actions.

The above provides a comprehensive overview into driver behavior, driver experience, amount and type of damage. It also tracks individual driver violations, enabling the operations team to identify trends and deliver customized training for drivers. The insights help in fostering a safer driving environment and optimizing overall fleet performance.

This dashboard offers a detailed analysis of Advanced Driver Assistance Systems (ADAS) performance. The data-driven insights help optimize vehicle safety features and enhance driver support, promoting safer and more efficient driving.

We are monitoring the performance of 36 ADAS fitted route buses. Four of them have camera mirrors. We get data from the telematics to compare and create algorithm based analytics

Predictive Analysis Application

The application above utilizes predictive analytics to forecast accident fault. It takes nine categorical features and predicts the outcome of who was at fault. The features are:

  • Light Condition
  • Weather Condition
  • Traffic Condition
  • Road Type
  • Accident Cause
  • Road
  • Collision Type
  • Damage Type
  • Location of Damage

It has been further enhanced to predict liability cost based on same features.

Conclusion

The Intelligent Accidents Monitoring and Management System represents a significant advancement in road safety technology, leveraging the power of machine learning (ML) and artificial intelligence (AI). By integrating advanced data visualization with predictive modeling, the system not only identifies patterns and trends in real-time accident data but also forecasts potential incidents. This AI-driven approach provides a comprehensive tool for proactively managing and mitigating the impact of road accidents, enhancing decision-making, and ultimately contributing to safer public transportion.
